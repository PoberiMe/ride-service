# Frontend

Frontend aplikacija za poberi.me.
Aplikacija je zgrajena z ogrodjem Angular in komunicira z zalednimi mikrostoritvami z REST API.

## Tehnologije
- Angular
- TypeScript
- HTML / CSS
- REST (HTTP)

## Zahteve
- Node.js (priporoƒçena LTS verzija)
- npm


## Zagon lokalno
npm install
ng serve

Aplikacija bo dostopna na:
http://localhost:4200

## Komunikacija z zaledjem
Frontend komunicira z naslednjimi mikrostoritvami:
- user-service
- ride-service
- route-service
